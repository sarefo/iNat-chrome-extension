Bulk Mode — How to Use

Select multiple observations and annotate them all at once.

Click Actions

ClickSelect or deselect an observation.
1–5 keysFocus an observation — shows the age/alive overlay immediately. Use zone keys to annotate. Press ↵ to open in a new tab (and deselect).
Shift (hold)While a card is focused, switches to the sex/evidence/mating overlay.
Shift+hoverShows the sex/evidence/mating overlay on mouse hover. Click a zone to annotate — without deselecting the observation.
  • ♀ Female / ♂ Male — sets the Sex annotation
  • 🏗 Construction — sets Evidence of Presence: Construction
  • 🥚 Egg — sets Evidence of Presence: Egg
  • ❤️ Mating — adds observation field "Behavior: mating = yes"

Toolbar Buttons

Select Page (p)Select or deselect all observations on the current page.
Select All (a)Select all loaded observations across all pages.
Add to Queue (x)Save the selection as a named queue for processing from the popup.
✕ ExitExit bulk mode and clear the session.
